The original FP32 FontCLIP ViT-B/32 image encoder producing 512-dimensional font embeddings.

The logical model ID is `fontclip-vit-b32`. Its technical release tag uses the
form `fontclip-vit-b32@<release-revision>`.

The release includes the approximately 352 MB ONNX model, attribute directions
regenerated against this FP32 artifact for all 37 attributes, the model API v1
manifest. Attribution and licensing information is documented in the
[repository README](https://github.com/MugiSus/fontcluster-models#attribution-and-licensing).
