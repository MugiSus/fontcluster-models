A compact MobileNet V4 Medium model producing 512-dimensional font embeddings.

The model ID and release tag are `mobilenet-v4-medium-v1`.

This is the default model downloaded on demand by FontCluster. The release
includes the ONNX model, its exact 37 attribute directions, and the model API v1
manifest. Attribution and licensing information is documented in the
[repository README](https://github.com/MugiSus/fontcluster-models#attribution-and-licensing).
